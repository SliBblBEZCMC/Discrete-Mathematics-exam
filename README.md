# Discrete-Mathematics-exam
Информация по билетам собрана на основе фотографий, которыми делились студенты. Так как сбор происходил в том числе по памяти, некоторые билеты могут быть представлены не в полном объёме или предствалены с иной формулировкой.
На данный момент известны 1 и 3 билет. Ответы немного отличаются от формулировки в лекциях, еще добавил пояснения для обычных людей.
